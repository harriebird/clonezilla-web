# Burn Clonezilla live CD

## Create Clonezilla live CD by InfraRecorder

To create bootable Clonezilla live CD on MS Windows with [InfraRecorder](http://infrarecorder.org/). Thanks to Victor Westmann for providing the screenshots.

1. Start the InfraRecorder program then choose `Actions` -> `Burn Image`:

   ![Select burn image option](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_01.png)

2. Choose the Clonezilla live iso image file:

   ![Choose the Clonezilla image file](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_02.png)

3. Choose the options to meet your requirement:

   ![Set burning options](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_03.png)

4. Start burning:

   ![Burning process start](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_04.png)

5. Burning process...:

   ![Burning process fill input buffer](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_05.png)

6. Burning process...:

   ![Burning process 35 percent](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_06.png)

7. Burning process...:

   ![Burning process 89 percent](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_07.png)

8. Burning process...:

   ![Burning process 100 percent](./images/burn-clonezilla-live-cd/burn_iso_clonezilla_08.png)

9. Burning process...:
